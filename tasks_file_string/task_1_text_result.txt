Завтра на большинстве территорий Украины ожидаются значительные осадки.
Об этом в Facebook рассказала синоптик Наталка Диденко.
"Похоже,  циклон Volker войдет в историю южного метеосериала.  именно он, по прогнозу, окажется завершающим
из   циклонов, которые организовали украинцам подзабытую зиму", - написала Диденко.
По ее словам, завтра  циклон  усложнит погоду в большинстве областей Украины.
